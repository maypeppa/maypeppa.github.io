搭建直播平台

在公众号看到 @陈利人 老师发了一篇 文章 说如何搭建自己的直播平台，里面还给了比较详细的步骤。

之前也好奇直播平台到底是个什么原理，就按照文章里面的步骤试了一把，代码放在了 github 上面。

默认的nginx发布包里面是没有带 rtmp 模块的话，所以还要自己编译一个有rtmp模块的nginx出来.

HLS最新实现对mp3格式 支持不好 ，所以使用文章里面的ffmpeg命令是没有audio的，换成aac格式就好了(-c:a libfdkaac)

hls.js 可以用来做播放器 watch it

#+title: 数据可视化 Data Visualization

- [[https://echarts.baidu.com/][echarts]] 百度开源的JS图表库。
- [[https://pyecharts.org/#/][pyecharts]] Python 是一门富有表达力的语言，很适合用于数据处理，当数据分析遇上echarts时，pyecharts 诞生了。
  - [[https://github.com/pyecharts/pyecharts/][github]] Pyecharts on Github
  - [[https://pyecharts.org/#/zh-cn/intro][document]] Get Started
- [[https://matplotlib.org/][matplotlib]] 一个Python 2D绘图库,配合mpl_toolkits.mplot3d可画出3D图形
- [[https://cs.stanford.edu/people/karpathy/convnetjs/][ConvNetJS]] 用于训练深度学习模型的JS库，可直接运行在浏览器。
